## 文档说明

个人还是比较喜欢JB全家桶。但在使用过程中遇到了一些问题，想找人寻求帮助，却发现资料太少。

于是，建了一个使用交流群，有在用的大佬可以加群，遇到问题可以一起讨论交流。

同时，也建议把遇到的问题，放到Issues，方便后来人。
